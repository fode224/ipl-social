## Description du projet
IPL Social est un projet développé dans le cadre d’un exercice DevOps pour créer une pipeline d’intégration continue (CI) avec GitHub Actions.  
Le projet est structuré en modules avec du code source dans le dossier `src/` et des tests automatisés dans le dossier `tests/`. La pipeline CI s’assure que tous les tests passent à chaque push ou pull request sur le dépôt, garantissant la qualité du code avant toute fusion.  

## Commande pour exécuter les tests
Avant de lancer les tests, installez les dépendances avec :

```bash
npm install
npm test
